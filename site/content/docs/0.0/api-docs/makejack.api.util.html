makejack.api.util

Helper functions to implement tools for makejack.

chmod

(chmod path-like mode)

Change file mode, given octal mode specification as string.

clj-source-file?

(clj-source-file? p)

Predicate for the given path being a clojure source file.

path

(path path-like)(path parent child)(path parent child & more)

Return a java.nio.file.Path, passing each argument to as-path.

Multiple-arg versions treat the first argument as parent and subsequent args as children relative to the parent.

(path->namespace path-like)
